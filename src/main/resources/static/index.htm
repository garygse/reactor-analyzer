Purpose

This tool aims to help provide a little more insight into what is happening under the covers when a Project Reactor reactive pipeline is running.

Note: this analyzer executes pipelines in order to inspect actual behavior, meaning it will not show events that do NOT happen. For example, it will not show any error handling capabilities of a pipeline if no error actually occurs during execution.

How to use the visualizer:

  1. Add Reactor Analyzer to your project, and use an Analyzer instance on your Flux or Mono pipeline.
  2. In the logs, look for the output of the analyzer and you should find some JSON.
  3. Copy that JSON into the box below and click Generate.
