Project Reactor Analyzer

Purpose

This visualizer provides a little insight into what is happening under the covers when a reactive pipeline is running in Project Reactor.

Note that this visualizer executes any provided pipeline, and attempts to visually explain the execution path taken. In other words, it will not show things that did NOT happen. For example, it will not show any error handling of the pipeline if no error actually occurs during execution.

How to use the visualizer:

  1. Add Project Reactor Analyzer to your project, and use an Analyzer instance on your Flux or Mono pipeline.
  2. In the logs, look for the output of the analyzer and you should find some JSON.
  3. Copy that JSON into the box below and click Generate.
